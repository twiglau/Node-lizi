# PM2配置和使用
- PM2 介绍
> PM2 nodejs 进程管理工具 (npm install pm2 -g)
> 守护进程: 服务挂掉之后自动重启
> 多进程: 更好的利用 CPU 和 内存
- PM2 配置
> 启动服务
> 常用命令
>> 
```
//查看
pm2 list
//重启
pm2 restart www(app name) | 0(id)
//停止
pm2 stop 0
//删除
pm2 delete 0
//启动
pm2 start 0
//进程,系统的信息
pm2 info 0
//日志
pm2 log www
//监控
pm2 monit www
```
